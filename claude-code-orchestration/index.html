Claude Code Orchestration

Optimize cost vs. quality tradeoffs when using multiple AI models in Claude Code. Route work to the right model — expensive reasoning for hard problems, cheap execution for boilerplate — so the total output quality stays high while spend drops. Fable 5 is the orchestrator; Opus, Sonnet, and Codex play supporting roles.
