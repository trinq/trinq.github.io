Context Management Strategies

Reference 0003 · Print-friendly cheat sheet · From Lesson 0010

The five strategies

StrategyWhat it doesBest forCost
Compaction Summarizes old turns, keeps signal Long multi-turn sessions Detail loss (lossy)
Observation masking Hides old tool outputs, keeps call records Heavy tool use (file reads, search) Zero extra compute
JIT retrieval Loads data on demand, keeps pointers Codebase exploration, debugging Extra tool calls
Sub-agent delegation Child explores, returns summary Research before coding Extra LLM calls
Structured note-taking Writes state to files, reads on resume Multi-session / multi-window tasks File management

Decision matrix

ScenarioPrimary strategySupporting strategy
Context at 80%+CompactionStructured notes (checkpoint first)
Agent reading many filesJIT retrievalObservation masking
"Understand the codebase"Sub-agent delegationJIT retrieval
AFK agent on scoped issueJIT + masking + notesCompaction (automatic)
Task spanning 2+ sessionsStructured notesCompaction within each session
Agent ignoring old constraintsJIT re-injectionMove constraints to CLAUDE.md top

Context window zones

UtilizationStatusAction
0–50%🟢 SafeFull reasoning capacity. Work freely.
50–80%🟡 WatchConsider compaction. Avoid large file reads.
80–100%🔴 DangerCheckpoint to progress file. Compact or restart.

Compaction priorities (ACON insight)

KeepDrop
Reasoning traces (why decisions were made)Raw tool outputs (file contents, logs)
Architectural decisionsRepetitive exchanges ("looks good" / "ok")
Unresolved bugs / open questionsExploration dead ends
Current plan / remaining workCompleted sub-tasks (record outcome only)

Daily hygiene checklist

  1. Monitor with /context before complex operations
  2. Use /clear between unrelated tasks
  3. Keep CLAUDE.md under 200 lines
  4. Put critical rules at the top of CLAUDE.md (primacy zone)
  5. Use sub-agents for research tasks
  6. Don't paste large files into chat — use tool-based retrieval
  7. Write PROGRESS.md before hitting 80%
  8. Restart aggressively — fresh session + good notes > degraded session
