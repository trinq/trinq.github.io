Agentic Engineering

An 11-lesson course on Matt Pocock's workflow for shipping software with AI agents. The harness, not the model, is the whole game. Built on queues, procedures, and code the agent can read.

Lessons

Lesson 0001 agentic-engineering

The Harness Is the Whole Game

Most of the leverage in AI-assisted engineering is in the harness, not the model. The harness is everything outside the model that shapes its behaviour — prompts, skills, CLAUDE.md, MCP servers, the codebase.

Lesson 0002 agentic-engineering

Procedures You Drive, Abilities You Lend

Distinguish procedure skills (you invoke them, you stay in control) from ability skills (the model invokes them when relevant). Get the frontmatter flag wrong and you'll leak control of your workflow to the model.

Lesson 0003 agentic-engineering

Queues, Not Loops

A while-loop running the agent forever sounds appealing but performs badly. Replace it with an issue-tracker queue: well-scoped items, mergeable on their own, item comes off when the PR merges.

Lesson 0004 agentic-engineering

DX and AX, or: Design for the Reader Who Reads Everything

DX is for the human reader. AX is for the agent reader. The overlap is huge — same rules (deep modules, clear interfaces, no deep nesting). Add top-of-file context blocks so the agent inherits what you'd otherwise explain verbally.

Lesson 0005 agentic-engineering

Grill Me, Then Scope It

The grill-me skill turns a vague request into a well-scoped queue item through 30–50 questions. Output ends with explicit Done means / Not done means lines. The highest-leverage skill in the workflow.

Lesson 0006 agentic-engineering

Write a PRD the Agent Can Read

A PRD is a 200–800 line decision-shaped file that captures the why (non-goals, rejected alternatives, decisions) the codebase can't. Six canonical sections. The agent reads it before touching code.

Lesson 0007 agentic-engineering

Create Issues, Atomic Work Units

Decompose a PRD into atomic GitHub issues, each sized to one mergeable PR. Vertical slicing is the default. Feature flags hide incomplete slices. Filed with gh CLI, not the web UI.

Lesson 0008 agentic-engineering

AFK Agents, the Loop Closes

An AFK agent reads the issue and PRD, works in a sandbox, opens a PR. A review agent in CI filters broken PRs. You review the survivors. All eight lessons form one chain: scope, document, decompose, execute.

Lesson 0009 agentic-engineering

Anatomy of the Agent Harness

The 12 components inside the box you've been calling "the harness." Three levels of engineering, the loop in motion, the co-evolution principle, seven decisions that define every harness.

Lesson 0010 agentic-engineering

Context Management: The Silent Killer

The #1 failure point in production agents, and five strategies to fix it. Context rot, ACON insight, context window zones, context hygiene for Claude Code.

Lesson 0011 agentic-engineering

Verification Loops: The 2–3× Quality Lever

The agent does something confidently wrong and nothing catches it. Fowler's guides vs. sensors, three verification types, Gather-Act-Verify cycle, the layering pattern.
