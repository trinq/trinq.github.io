Agent Skills — Writing Great Skills

A four-axis course on writing great agent skills, based on Matt Pocock's "Building Great Agent Skills: The Missing Manual". Trigger · Structure · Steering · Pruning.

Lessons

Lesson 0001 agent-skills 2026-07-01

Trigger — Who fires the skill?

User-invoked vs model-invoked. Two mechanisms, two costs. The bug class, not the cost, is the real question.

Lesson 0002 agent-skills 2026-07-01

Structure — Steps and reference, nothing else

Two units only: steps and reference. Push branch-specific material behind context pointers. Keep skill.md minimal.

Lesson 0003 agent-skills 2026-07-01

Steering — Leading words and leg work

Pack meaning into a term the agent already half-knows. Repeat it. Split a skill when the agent rushes toward a future goal it shouldn't see.

Lesson 0004 agent-skills 2026-07-01

Pruning — Sediment, crud, and no-ops

Three failure modes. The deletion test is the one technique that catches no-ops. Single source of truth keeps it from coming back.

Lesson 0005 agent-skills 2026-07-01

The Skill Checklist, retold

The whole framework as one essay. Trigger · Structure · Steering · Pruning, in order, with citations to the original talk.
