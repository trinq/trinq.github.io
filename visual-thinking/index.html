Visual Thinking

Cognitive tools for agent cognition — based on Prof. Judy Fan's MIT BCS Colloquium (March 2025). When should an agent render a diagram instead of prose? How do VLMs really compare to humans on plot reading? Where does your chart-pick logic fall into the bar-plot-purist trap?

Lessons

Lesson 0001 visual-thinking

What is a cognitive tool?

The number line, Cartesian coordinates, drawing. Frames the rest of the course around Fan's question: what new question does this representation make answerable?

Lesson 0002 visual-thinking

Visual abstraction: detail scales with context

The drawing-game study. Humans sparsify deliberately based on the lineup. CLIPasso does not. The dissociable vs. cumulative hypothesis for explanation vs. depiction.

Lesson 0003 visual-thinking

VLMs read plots wrong (even when they're accurate)

Fan's six-test benchmark against BLIP-2, LLaVA, GPT-4V. Why accuracy isn't enough. The audience-sensitive hypothesis for chart selection. Brockbank's measurement critique.
