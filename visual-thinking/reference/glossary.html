Glossary

Vocabulary for Fan's cognitive-tools frame, anchored to the talk and ready for agent-design conversations.

The frame

Cognitive tool
Fan's term for an external representation (number line, coordinate system, sketch, diagram, chart) that changes what its user can think about. The defining test: does this representation make a new question answerable? See Lesson 0001. Talk timestamp: 03:52–13:00.
Closing the loop
Fan's research-program motto: humans don't just discover useful abstractions, they apply them to make new things. The discovery-and-engineering loop is the throughline. Agent equivalent: parse a chart → use what you parsed to act.
Visual abstraction
The deliberate reduction of visual detail to highlight what is relevant. Two forms: sparsification (fewer strokes) and selective omission (drop the background). Sparsification is one knob; selective omission is a different one.
Production budget
The constraint on how much an artist can produce — time, strokes, ink. SEVA uses four budgets. At the tight end, humans and CLIPasso diverge. See Lesson 0002.

Drawing (Part 1)

Depiction
Drawing intended to communicate what something looks like. Faithful to surface. Preserves background and non-causal structure. Huey: depicted drawings win on object-identification tasks.
Explanation
Drawing intended to communicate how something works. Suppresses surface. Privileges causal parts, motion lines, arrows. Huey: explanation drawings win on action-understanding tasks.
Cumulative hypothesis
The hypothesis that explanations are depictions + extra mechanism (the same drawing, with more arrows). Rejected by Huey's data — if cumulative were true, depictions should be at least as good as explanations on every task. They are not.
Dissociable hypothesis
The hypothesis that explanations and depictions are different drawings, with different vocabularies. Causal parts are drawn in both, but the relative weights differ. Supported by Huey's data. Agent implication: an explanation prompt should not start from a depiction and add arrows.
Referential context
The lineup of alternative objects the viewer is choosing among. The drawing-game studies show that the lineup — not the target alone — determines how much detail a human will produce.
SEVA
Sketch-ELaborated Visual Abstraction benchmark. 90,000 sketches from 5,500 people of 128 concepts at four production budgets. Fan-lab, 2024. Use for: stress-testing any sketch-generation system against human sparsification behavior.
CLIPasso
A neural sketch-generation algorithm from Fan's collaborators (Winkler et al.). Generates vector sketches with controllable stroke budget. The comparison point for SEVA — recognizability matches humans at matched budget, but the meaning distribution diverges at the tight end.
Stroke-budget sparsification
The behavior of reducing detail as the budget tightens. CLIPasso sparsifies differently from humans. The lesson is: "less detail" is not the same thing as "lower fidelity."
Meaning distribution
The full distribution of labels a sketch evokes when shown to viewers. Not just the top label — the entropy and the semantic-neighborhood structure. CLIPasso and humans match on the top label; they diverge on the distribution.

Data viz (Part 2)

Playfair 1786
William Playfair's Commercial and Political Atlas — the first time-series line chart ever made. Kicker for the data-viz half of the talk: like the number line, the time-series line chart was invented, not discovered.
VLM
Vision-Language Model. A multimodal model that ingests images (and text) and produces text. The class Fan + Verma benchmarked: BLIP-2, LLaVA, GPT-4V, Matcha, Pix2Struct, specialized variants. Note: "VLM" as a term is overloaded in 2024–25 to also mean "video-language model" — Fan uses the image-text sense.
GGR
Graphic Gestalt Reasoning. One of six plot-reading tests in the Verma benchmark. Tests basic visual reasoning over charts.
VLAT
Visual-Literacy Assessment Test. A commonly used chart-reading instrument. Used in Brockbank's measurement analysis.
CALVI
Critical-thinking Assessment of Literacy in Visualization. Adversarial plots — designed to trick with misleading axis scales. The test where the VLM accuracy gap is largest. See Lesson 0003.
HOLF / HOLF-multi
Higher-Order Learning Framework. Chart-reading test variants. Less commonly cited by name in industry evals; Fan's lab uses them.
Chart-QA
A standard ML benchmark for chart-question answering. The benchmark Fan argues is flattering — it shows the smallest accuracy gap between humans and VLMs, but the gap widens dramatically on CALVI-style adversarial items.
Human-noise ceiling
The expected band of human-vs-human divergence on a chart-reading test — how much two random humans would disagree. Models falling below this band on wrong-answer divergence are failing in non-human ways, even if their accuracy is high.
Audience-sensitive hypothesis
Huey's plot-selection proposal: people pick plots based on what features of the plot are needed to answer the question, not based on chart-type prior. Supported by Fan's data on plot-choice distributions. Agent implication: agents that pick charts should rank by question-relevance, not chart-type preference.
Bar-plot purist
The hypothesis (rejected) that humans default to bar plots regardless of the question. Used as a strawman in Huey's analysis. Agent trap: an agent that defaults to "always render a bar chart" is a bar-plot purist. Don't be that agent.
Factor structure
The latent skill decomposition that explains error patterns on a chart-reading test. Brockbank shows the textbook ontology (find-max, identify-cluster, etc.) is not the right factor structure — a four-factor alternative predicts errors much better. Agent implication: don't trust the textbook decomposition for prompt engineering.

Frame for agent design

Question-shaping
The principle that an artifact (drawing, chart, diagram) is shaped by the question that follows it, not by the object alone. The cognitive-tool frame applied to output generation.
Explanation-prompt
An agent prompt structured for the dissociable hypothesis: (a) name the causal structure first, (b) suppress non-causal parts, (c) privilege arrows and motion lines. Not "depiction + extras."
Chart-pick rubric
A three-step process for an agent that picks a chart: (1) name the question, (2) name the features of the chart needed to answer it, (3) pick the chart with those features. Closes the bar-plot-purist trap.
Error-pattern check
An evaluation method that compares the wrong-answer distribution of an agent against the wrong-answer distribution of a human baseline, not just the accuracy number. The right number to look at when "Chart-QA says 92%" but you don't trust it.
