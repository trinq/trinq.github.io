Cognitive-tool taxonomy

Fan's three-behavior ladder — perception → production → communication — and the agent-design corollaries. One-page cheat sheet. Print-friendly.

Around 11:00 of the talk, Fan lays out a three-rung ladder of visual cognition. Each rung builds on the previous one. Each has an agent-design corollary that lands cleanly. Memorize the rungs; the corollaries fall out.

The ladder

RUNG 1
Visual perception
Transform raw sensory input into semantically meaningful perceptual experience. The pre-requisite for anything else. Talk: 11:00–11:40.
↓
RUNG 2
Visual production
Generate a set of markings that leave a meaningful, visible trace in the physical environment. Talk: 11:40–12:10.
↓
RUNG 3
Visual communication
Decide how to arrange the graphical elements and in what order to have a particular impact on other minds. Talk: 12:10–13:00.

The progression matters. You can't communicate with a drawing you can't perceive. You can't communicate a drawing you can't produce. The hard rung — and the one Fan spends 80% of the talk on — is rung 3: communication. That's where context, intent, and audience enter.

Mapping to agent behavior

Each rung has an agent-design reading. They're corollaries, not equivalences — agents don't "perceive" in Fan's sense, but the functional mapping is useful.

Rung Human behavior Agent behavior Failure mode
1 — Perception See a bird; recognize it as a robin Encode an image; identify what's in it VLM misclassifies; CLIP-style aliasing
2 — Production Move a pen; leave a mark Emit a sketch, diagram, or chart CLIPasso-style sparsification that loses meaning distribution
3 — Communication Decide how much detail, which vocabulary, which level Decide what shape of artifact matches the user's question Bar-plot purist; depiction-when-explanation-needed; budget-sparsification without question-relevance

The recurring theme: most agent failures on visual artifacts are rung-3 failures, dressed up as rung-1 or rung-2 failures. The model sees fine. The model produces fine. The choice of what to produce is wrong.

The "closing the loop" extension

Fan's research program extends the ladder into a fourth concept: closing the discovery-and-engineering loop.

EXT 1
Discovery
Find an abstraction that explains how the world works.
↓ ↓ (the loop)
EXT 2
Application
Apply the abstraction to make new things. (Darwin → land management. Feynman diagrams → QED calculations. Playfair → economic policy.)

For agents: an agent that only "discovers" (parses the chart) but never "applies" (acts on what it parsed) has half-closed the loop. A chart-reading agent that doesn't use the chart for a downstream decision is doing Fan-style discovery in isolation. That isn't cognitive-tool behavior. It's a perception exercise.

One-line summary

Perception → Production → Communication. The cognitive-tool frame lives on rung 3. The closing-the-loop frame adds Application. Most agent failures are rung-3 failures in disguise.
