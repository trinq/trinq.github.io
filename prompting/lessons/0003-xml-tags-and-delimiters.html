Lesson 0003: XML Tags & Delimiters

Cách tổ chức prompt để Claude hiểu rõ nhất

Tại sao cần Delimiters?

Trong video, Hannah nói: "Claude really loves structure, loves organization." Delimiters là cách bạn cung cấp structure đó.

Delimiters giúp Claude phân biệt: thông tin tĩnh (system) vs dữ liệu động (user) vs metadata vs examples.

So sánh: Không dùng vs Dùng delimiters

Không dùng delimiters
Here's a form to analyze:
Vehicle A checkbox 1: checked
Vehicle B checkbox 12: checked
And here's a sketch of the accident...

Please determine who is at fault.

Claude phải đoán đâu là instructions, đâu là data.

Dùng XML tags
<task>
Analyze this accident form and determine fault.
</task>

<form>
[form data here]
</form>

<sketch>
[sketch data here]
</sketch>

Claude hiểu ngay: đây là task, đây là form, đây là sketch.

3 Loại Delimiters

1. XML Tags (Khuyên dùng)

Claude được fine-tune trên XML → hiểu rõ nhất.

<task> What Claude should do </task>

<form> User's input data </form>

<example> Input → Output pairs </example>

<history> Conversation context </history>

Best practice: Đặt tên tags theo ý nghĩa, không phải vị trí. VD: <form> thay vì <input1>

2. Markdown Headers

OK cho CLI tools như Claude Code, nhưng không parse được dễ bằng XML.

## Task
Analyze this form...

## Form Data
[content here]

## Examples
[examples here]

3. Quotation Marks / Separators

Ít reliable nhất. Chỉ dùng khi không còn cách nào khác.

---
CONTEXT:
[content]
---
INPUT:
[content]
---
EXPECTED OUTPUT FORMAT:
[format]

Pattern: System Prompt với XML Structure

<role>
You are an AI assistant helping a claims adjuster
review Swedish car accident forms.
</role>

<instructions>
- Stay factual. If uncertain, say so.
- Examine the form first, then the sketch.
- Wrap your verdict in <verdict> tags.
</instructions>

<form_structure>
The form has:
- Title
- Two columns (Vehicle A, Vehicle B)
- 17 rows describing incident details
- May contain checkmarks, X marks, or circles
</form_structure>

<examples>
<example>
  <input>[example form]</input>
  <output>Vehicle B is at fault because...</output>
</example>
</examples>

Exercise

Viết lại prompt bạn đang dùng với XML structure:

  1. Identify: đâu là system context, đâu là user input, đâu là examples
  2. Wrap mỗi phần trong XML tags có ý nghĩa
  3. Test: prompt này có chạy đúng không?
