Design with AI — Glossary

Key terms and concepts for human-AI collaborative design

Core Concepts

Taste

The subjective quality of design judgment — what feels original, meaningful, and genuinely good. AI cannot generate taste, only execute rules.

Design Rules

Objective principles AI can execute: spacing, typography, color theory, hierarchy, alignment. These are learnable and measurable.

Taste Gap

The difference between AI's ability to execute design rules and its inability to understand taste. Every method is a workaround for this limitation.

Design Skill Method 1

A prebuilt prompt or workflow that adds design intelligence to an AI tool. Examples: Impeccable, TasteSkill, Emil Kowalski's UI skills. Fastest approach, decent results.

Component-by-Component Method 2

Building design piece by piece rather than all at once. Slowest approach but highest quality. Human defines meaning, direction, and references; AI executes each component.

Inspiration Board Method 3

Curating a set of reference images and examples to give AI clear visual direction. Balanced approach between speed and quality.

Workflow Phases

Start with Meaning

Defining the foundation: who is this for, what problem does it solve, what should it feel like. Must be human-defined before AI execution.

Reference Collection

Gathering design examples to study and draw inspiration from. Should be organized by category (navigation, hero, cards, etc.).

Structure Mapping

Defining what components are needed for a project (e.g., website: nav, hero, features, pricing, CTA, footer).

Asset Generation

Using AI image/video tools to create custom visuals that match the design direction. Key tools: GPT Images 2.0, Seedream 5.0.

Refinement Loop

Iterating on AI output to fix obvious issues and add polish. The human reviews and corrects; AI executes changes.

Tools

Design Skill

A specialized prompt or workflow plugin that adds design intelligence to an AI tool. See: Impeccable, TasteSkill, Skills.sh.

Cursor

AI-first code editor. Primary tool for building designs from prompts, especially for web/app development.

Codex

CLI-based AI coding agent from OpenAI. Good for longer conversations and multi-file changes.

Mobbin

Design reference platform showing real product screens. Primary source for studying actual UI patterns.

Awwwards

Curated website showcase featuring high-end web design. For high-end web inspiration.

Quiver

SVG generation tool. SVGs allow better animations because elements are separate.

Layering

Using background removal and compositing to add depth, motion, and interaction to designs.

Design Categories

Hero Section

The primary visual area at the top of a page. Components: headline, description, CTA, visual.

Cards

Self-contained UI elements that group related information. Used for features, pricing, testimonials.

Empty States

UI shown when no data is present (empty cart, no search results). Opportunity for personality and guidance.
