Asset Generation with AI

Lesson 0007 · ~8 minutes · Skill

The fastest way to differentiate a design is custom imagery. AI tools now let you generate unique assets in minutes. This lesson covers the workflow, the tools, and how to prompt them well.

The Difference
Generic products use stock photos. Distinctive products use custom-generated assets. AI lets you create the distinctive version — at scale.

The Four Asset Categories

Different assets need different tools. Understanding which to use when is the first skill.

Custom Images
Photorealistic or stylized product photography, hero imagery, illustrations
Use: GPT Images 2.0, Seedream 5.0 (or similar generative models)
Logos
Brand marks, wordmarks, monograms
Use: Quiver, Ideogram, or dedicated logo generators
Vectors & Icons
UI icons, diagrams, custom illustrations in SVG
Use: Quiver (SVG generation) or dedicated vector tools
Motion & Video
Product demos, background loops, social content
Use: Google Flow, Runway, Pika

The Asset Generation Workflow

Identify what you need

Be specific: a hero illustration? A set of 12 feature icons? A product demo reel?

Find visual references

Use your reference library (Lesson 0005). Pick 2-4 examples that match the direction.

Write a detailed prompt

Apply the prompt formula (Lesson 0006). Include branding context, constraints, and feeling.

Generate multiple variations

Don't stop at one. Generate 4-8 versions and pick the strongest.

Refine the winner

Use focused refinements: lighting, color palette, composition, mood.

Edit for cohesion

Adjust colors, cropping, and treatment so assets feel like a set.

Custom Image Prompts

For AI image generation, prompts need different specifics than UI prompts. Use this structure:

Hero illustration for a productivity app. Scene: A minimalist workspace at dawn, soft natural light streaming through a large window. Desk surface barely visible, mostly negative space. Mood: Calm focus. Inviting but not cozy — more aspirational than comfortable. Style: Soft 3D rendering, muted color palette (warm off-white, soft terracotta, deep navy accents). Influences: Stripe, Linear, modern Apple marketing. Format: 16:9, ultra-wide composition suitable for a hero section. Do not include: text, people, specific products, busy details.

Note what works: specific scene, clear mood, palette reference, format defined, exclusions explicit.

The Branding Lock

Custom assets must feel like part of your brand, not strangers in your layout. Always lock in:

  • Color palette — Same hex codes as your design system
  • Lighting tone — Warm vs. cool, dramatic vs. soft
  • Rendering style — Flat, gradient, 3D, photorealistic
  • Composition — Minimal vs. complex, negative space ratios

When these align, assets integrate seamlessly. When they don't, the page feels disjointed.

SVG & Icon Strategy

Custom icons can completely change the feel of a product. Most products use icon libraries; distinctive products use custom-drawn or generated sets.

For icon generation with AI:

  • Generate a style test first — one icon — and iterate until the style is right
  • Then generate the full set using that locked style as reference
  • Edit in vector software if needed for consistency tweaks
  • Keep stroke width, corner radius, and fill treatment consistent across the set
The Style Lock First
For any asset set, perfect one example first. Then generate the rest using it as the reference. Consistency comes from referencing an existing example.

Video & Motion

Video generation is the newest frontier. Current tools work best for:

  • Short loops (5-15 seconds) for hero backgrounds
  • Product showcase clips (rotating, zooming)
  • Atmospheric backgrounds (subtle motion, abstract)
  • Social content snippets

Approach video prompts like image prompts, but add:

  • Motion description ("slow camera push-in," "subtle parallax")
  • Duration ("5-second loop")
  • Ending condition ("fade to white," "loop seamlessly")

Common Asset Mistakes

  • Generating without references — Always start with visual references in your library
  • Stopping at first output — Generate multiple variations; the first rarely wins
  • Ignoring style consistency — Lock the style early or your assets will look like strangers
  • Over-relying on AI — Sometimes a human-designed icon or illustration beats AI-generated ones. Know when to step in.

Exercise: Generate a Cohesive Set

Practice · 30 minutes
Create 3 hero illustrations that feel like a set.

Use the product idea from previous lessons.

Step 1: Define your locked style: palette, rendering style, lighting tone.

Step 2: Pick 2-3 reference images for your scene direction.

Step 3: Generate 4-8 versions of the first scene.

Step 4: Pick the winner. Note what worked.

Step 5: Generate the next 2 scenes with the winner's style as reference.

Step 6: Edit in any tool needed to make all 3 feel cohesive.

Save these — they pair with your UI components from Lesson 0004.

Key Takeaways

  • Custom assets differentiate from stock-photo competitors
  • Different tools serve different asset categories
  • Always generate multiple versions; pick the strongest
  • Lock style early; reference the winner for subsequent assets
  • Branding lock: palette, lighting, style, composition must align
