Lệnh cơ bản

/ponytail full    # Bật khi agent propose code — chặn code thừa
/ponytail-review  # Review diff sau khi code xong

7 Rung Ladder

Rung Check Nếu có
R1 YAGNI? Reject — không cần
R2 Reuse? Dùng lại code cũ
R3 stdlib? Dùng thư viện có sẵn
R4 native? Dùng platform API
R5 one-liner? Inline đủ
R6 minimum? Giữ code tối thiểu
R7 delete? Xóa hoàn toàn

Combined Workflow

Grill → PRD → Issues → Implement → Review → Merge
                    ↑
              /ponytail full
              Viết test (TDD) trước

Questions to ask

  • Đã cần feature này chưa? (R1 YAGNI)
  • Code tương tự đã có chưa? (R2 reuse)
  • Thư viện stdlib đủ chưa? (R3 stdlib)
  • Platform API đủ chưa? (R4 native)
  • Có inline được không? (R5 one-liner)
  • Cắt bớt được không? (R6 minimum)
  • Có thể xóa không? (R7 delete)
