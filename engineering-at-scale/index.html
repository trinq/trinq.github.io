All Lessons

Ten lessons from Spotify's VP of Engineering on how a hyperscaler thinks about specs, code review, ownership, and dependencies — translated to a small team's scale. Anchored to Beyond Code #12 with Niklas Gustavsson.

Lessons

Lesson 0001 engineering-at-scale 2026-06-30

Prototype, Don't PRD

Replace a static PRD with a working prototype, ship it to a real user for feedback, then write the formal spec.

Lesson 0002 engineering-at-scale 2026-06-30

Hyrum's Law Hits Your Agents

Learn Hyrum's Law and find the parts of your codebase that AI agents will silently corrupt during migrations.

Lesson 0003 engineering-at-scale 2026-06-30

Review the Script, Not the PR

Stop reviewing changes line-by-line. Review the script that made the changes. Spotify's 85%-auto-merge trick at any scale.

Lesson 0004 engineering-at-scale 2026-06-30

Measure the Aggregate

Track three metrics weekly: regressions caught, generator runs without regressions, prototype-to-ship ratio.

Lesson 0005 engineering-at-scale 2026-06-30

When to Auto-Merge

Tier every PR into three levels: auto-merge, light review, or full review. 80% should auto-merge.

Lesson 0006 engineering-at-scale 2026-06-30

Incident Response at Scale

Rollback in the first 30 minutes. Three-lens postmortem (system, process, human). Severity tiers and runbook discipline.

Lesson 0007 engineering-at-scale 2026-06-30

Code Ownership at Scale

Write a CODEOWNERS file with an explicit default. One-page runbook per critical module. Don't retrofit ownership after N=3.

Lesson 0008 engineering-at-scale 2026-06-30

Dependency Management

Four-tier dep policy, five-question decision framework. Prefer narrow APIs. Automate updates with Dependabot.

Lesson 0009 engineering-at-scale 2026-06-30

Tech Debt and Pruning

Four verdicts (keep, hold, prune, rewrite). Three cadences (PR-time, monthly, quarterly). Git has history; delete the code.

Lesson 0010 engineering-at-scale 2026-06-30

Team Growth

Workflow composes from N=1 to N=10+. N=3 is the load-bearing milestone. Set structure before you grow.
