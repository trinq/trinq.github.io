Three Methods Cheatsheet

Leon Lin's three methods (per your summary; the original tweet was not retrieved at design time) and when each one wins.

MethodSpeedQualityDistinctivenessBest fit
1. Design SkillVery fastMediumLow–MediumSmall projects, prototypes, time-bound
2. Component-by-ComponentSlowHighestHighProducts that need to stand out
3. Inspiration BoardBalancedHighMedium–HighMid-complexity, balanced speed/quality

Decision tree

If the page needs to ship tomorrow → Method 1 (Design Skill).
If the page needs to be unmistakably yours → Method 2 (Component-by-Component).
If the page needs to be both fast and distinct → Method 3 (Inspiration Board).

Anti-patterns

  • One-shot prompt: "make a modern, clean landing page." Always sloppy.
  • "Make it pop" — undefined; require the user to name the audience feeling.
  • Default gradients on every section — looks 2023.
  • No reference attached to a component — implies no intent was written.
